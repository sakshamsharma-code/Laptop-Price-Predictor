About This Project

This project is a Laptop Price Predictor that uses Machine Learning to estimate the market price of a laptop based on its specifications.

It considers hardware features like brand, processor, RAM, storage (SSD/HDD), graphics card, and software features like operating system, MS Office availability, and more.

It is trained on a curated dataset of laptops available in the market, and helps users quickly understand what price range their laptop might fall into â€” whether buying or selling.

Built using Flask, Python, and Bootstrap, this app provides an intuitive UI for users to input data and see real-time predictions.
